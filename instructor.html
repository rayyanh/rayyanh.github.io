What is InstructorHQ?

InstructorHQ is a website that is an online instruction platform. It is made for both students as well as teacher management. The software is based on an online web app that allows for both students and teachers to have a management platform for all their courses across the web.

Project Goals & Process

As part of a class project, the goals were given to us. In the simplest terms, the goal was to create a better user experience as well as UI for this platform. It was to be done using the user experience process and then finally submitting wireframes and high fidelity mockups.

Challenges

When doing competitor research, it was difficult to find similar platforms like the one that we were to create a clean UI for. It was also difficult to create everything in general as it was a very niche kind of web application. Fortunately, the class instructors guided me through the process.

Objective

To make the platform have a better User Experience

Although this was very broad, it gave me and the team wiggle room as we worked closely to the CEO. (We were the first batch of interns at Kadama)

Persona(s)

Persona is listed on the behance page linked below.

Solution

The solution was to keep doing research until finding a similar product and to do the right user research. Reaching out to colleagues and working on 'design reviews' was helpful and ultimately helped me achieve my goal in terms of this project.
